# ADB Commands

```bash
### adb commands
adb devices                            	        // list running emulators
adb -s 127.0.0.1:6555 shell
adb -s 127.0.0.1:6555 pull path/to/file.ext ./  // pull file from the emulator 
adb root
abd install 				        // install apps 
adb log											// Android log
```
